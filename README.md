Ejecición del **algoritmo PI-Parte 1**:
``` bash
$ cd Python/
$ python3 algoritmo-v3.py
```

Ejecución del **algoritmo PI-Parte 2**:
``` bash
$ cd Python/
$ python3 tesis.py
```
Ejecución del software **Petrinator**:
``` bash
$ cd Redes_de_Petri/
$ ./Petrinator.sh
```
